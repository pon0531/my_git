●確定病名/ Disease Name：文字，均為「猴痘」。
●發病年份/ Year_Onset：西元年數字，發病日所屬年份。 
●發病月份/ Month_Onset：月份數字，發病日所屬月份。 
●縣市/ County_living：文字，病例居住地縣市。 
●鄉鎮/ Town_living：文字，病例居住地鄉鎮市區。 
●性別/ Gender：類別文字：F(女)、M(男)、X(第三性)、U(不詳)、否。 
●年齡層/ Age_Group：文字，描述年齡區間：未滿5歲者以1歲為1區間；5~69歲者以5歲為1區間；70歲以上為1區間。 
●確定病例數/ Number_of_confirmed_cases：數字(整數)
